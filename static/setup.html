Integration Setup

Connect RetainAI to your live application.

1 Install Frontend Widget

Paste this into the <head> of your website.

Copy
<script src="https://churnkey-demo.onrender.com/static/widget.js"></script>
<script>
  RetainAI.init({ 
    projectId: 'CLIENT_ID',
    color: '#2563EB' // Your Brand Color
  });
</script>

2 Configure Stripe Webhook

In Stripe Dashboard, go to Developers > Webhooks and add this URL:

https://churnkey-demo.onrender.com/webhook

Listen for: invoice.payment_succeeded
